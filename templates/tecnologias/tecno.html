Python

Es un lenguaje de programación de tipado dinamico, multiproposito y orientado a objetos. Su popularidad ha aumentado un 30% en el 2021 al tener una sintaxis limpia y sencilla produciendo codigo más legible, una curva de aprendizaje ¡rapida! para que hasta niños de primaria escriban codigo Python.

Python es el segundo lenguaje de programación más utilizado en el mundo según una Encuesta realizada en Stack Overflow a más de 80,000 desarrolladores en Mayo 2021 a sido un lenguaje creado en la decada de los noventa por el desarrollador Guido van Rossum el cual no se imagino que este lenguaje iba a ser tan demandado y popular. Python es utilizado en:

  • Ciencia de Datos
  • Desarrollo Web
  • Desarrollo Movil
  • Aplicaciones de Escritorio
  • Seguridad en informatica
  • Inteligencia Artificial
  • Machine Learning

Las distribuciones GNU Linux utilizan python para su funcionamiento teniendo de forma predeterminada interpretes de Python incoporados. Cada distro linux cuenta con su propio gestor de paquetes, fedora utiliza el administrador "dnf" que a sido escrito en el lenguaje python aunque su velocidad de ejecucion no sea la mejor pero no le resta medito al poder que fue adquiriendo a lo largo del tiempo contando con PIP un administrador de paquetes para instalar multiples librerias y frameworks.

Python cuenta con una gran comunidad activa que a desarrollado sus propias librerias, cada una fue evolucionando en versiones sin embargo Python permite trabajar con diferentes versiones de una misma libreria en más de un proyecto a través de "entornos virtuales" las librerias se instalan en el entorno del proyecto y no en todo el sistema opertativo evitando las dependencias de un proyecto interfieran en otro.
