¿Qué es un CRUD?

En programación se le conoce como "CRUD" a la interacción con una base de datos a través de peticiones HTTP realizadas por un lenguaje de programación para insertar, modificar, leer y eliminar datos de un registro. La base datos puede ser relacional (SQL) y no relacional (NoSQL) ambas poseen objetivos diferentes pero tienen la misma meta recibir, almacenar y enviar datos.

Una base datos sql gestiona los datos por medio de tablas que se van relacionando, las consultas se realizan con el lenguaje SQL y una base datos NoSql esta basada en tecnologia JavaScript por lo que los datos se almacenan en documentos que tienen la estructura de un archivo JSON o de un arreglo asociativo, las consultas se realizan por comandos y objetivos. En la actualidad las bases de datos poseen una interfaz grafica que facilita la manipulación de los datos.

Significado de las siglas CRUD

Siglas Significado Sentencia SQL
C Create (Insertar) INSERT
R Read (Leer) SELECT
U Update (Actualizar) UPDATE
D Delete (Eliminar) DELETE
